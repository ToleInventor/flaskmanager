Flask Manager

Automated Flask project management for Windows developers
Operating System Detection

Installation Instructions

Step 1: Download the installer
Click the download button above to save installer.ps1 to your computer.
Step 2: Run the installer
Right-click on installer.ps1 and select "Run with PowerShell".
If a security warning appears, click "More info" then "Run anyway".
The installer will automatically request administrator privileges.
Step 3: Follow the prompts
The installer will:
- Create C:\Program Files\FlaskProjectManager
- Download the Flask Manager executable
- Add the program to your system PATH
- Create a Start Menu entry
Administrator privileges required
The installer needs administrator access to write to Program Files and modify the system PATH. The script will automatically request elevation via UAC.

How to Run Flask Manager After Installation

Method 1: Windows Search (recommended)
Press the Windows key, type "Flask Manager", and press Enter.
Method 2: Command Prompt
Close all Command Prompt windows, open a new one, and type:
flask_manager
Method 3: Run directly
"C:\Program Files\FlaskProjectManager\flask_manager.exe"

Manual PATH Configuration

If the installer did not add Flask Manager to your PATH, or if you want to set it up manually, use one of the following methods:

Option A: Add to System PATH (Recommended)

Run this command in PowerShell as Administrator:

[Environment]::SetEnvironmentVariable("Path", [Environment]::GetEnvironmentVariable("Path", "Machine") + ";C:\Program Files\FlaskProjectManager", "Machine")

Option B: Create a Permanent Alias (User Level)

This creates a PowerShell profile with a flask_manager function that works like a native command:

$profileDir = Split-Path $PROFILE -Parent; if (!(Test-Path $profileDir)) { New-Item -ItemType Directory -Path $profileDir -Force }; Add-Content $PROFILE "`nfunction flask_manager { & 'C:\Program Files\FlaskProjectManager\flask_manager.exe' $args }"

Option C: Add to User PATH via Command Prompt

Run this in Command Prompt as Administrator:

setx PATH "%PATH%;C:\Program Files\FlaskProjectManager"
Important: After running any PATH modification command, you must restart your Command Prompt or PowerShell window for the changes to take effect. A full system restart is recommended to ensure all applications recognize the new PATH.

First Time Usage

When you first launch Flask Manager, you will be asked to choose between:

  • Current user - Projects stored in %USERPROFILE%\flask_manager
  • All users - Projects stored in C:\Users\flask_manager

The manager will verify that Python and Visual Studio Code are installed. If missing, it will guide you through the installation process or open the download pages.

System Requirements

  • Windows 10 or Windows 11
  • Internet connection (for initial download)
  • Administrator privileges (for installation only)
Python and VS Code
Flask Manager requires Python and Visual Studio Code. If not found, the application will redirect you to the official download pages.

Uninstalling

To completely remove Flask Manager from your system, run this in PowerShell as Administrator:

Remove-Item -Path "C:\Program Files\FlaskProjectManager" -Recurse -Force

Then manually remove C:\Program Files\FlaskProjectManager from your system PATH via Environment Variables.
