Flask Manager

Automated Flask project management for Windows

Flask Manager is a command-line tool that automates the creation, management, and execution of Flask web applications on Windows. It handles virtual environments, dependency installation, and project scaffolding.

Operating system detection

Installation instructions

Step 1
Click the download button above to save installer.cmd to your computer.
Step 2
Locate the downloaded file (usually in your Downloads folder).
Right-click on installer.cmd and select "Run as administrator".
Step 3
Follow the on-screen prompts. The installer will:
- Download the Flask Manager executable
- Install it to C:\Program Files\FlaskProjectManager
- Add the program to your system PATH
- Verify Python and VS Code installations (install if missing)
Administrator privileges required
The installer must run with administrator privileges to write to Program Files and modify the system PATH. Right-click the file and select "Run as administrator".

How to run Flask Manager after installation

Option 1: Command Prompt (recommended)
Open any Command Prompt window and type:
flask_manager
Press Enter. The manager dashboard will appear.
Option 2: Run dialog
Press Windows + R, type flask_manager, and press Enter.
Option 3: Start menu
Type "Flask Manager" in the Windows Start menu search and click the application.

First time usage

When you first launch Flask Manager, you will be asked to choose between:

  • Current user - Projects stored in %USERPROFILE%\flask_manager
  • All users - Projects stored in C:\Users\flask_manager

The manager will also verify that Python and Visual Studio Code are installed. If missing, it will guide you through the installation process.

System requirements

  • Windows 10 or Windows 11
  • Internet connection (for initial download and Python/VS Code installation)
  • Administrator privileges (for installation only)

Uninstalling

To completely remove Flask Manager from your system:

Run as Administrator: rmdir /s /q "C:\Program Files\FlaskProjectManager"

Then manually remove C:\Program Files\FlaskProjectManager from your system PATH via Environment Variables.
